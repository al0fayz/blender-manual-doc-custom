Introduction¶

The Graph Editor allows users to adjust animation curves over time for any animatable property. F-Curves.

The Graph Editor.¶

Main Region¶

The curve view allows you to view and edit F-curves. An F-curve has several key parts:

Curve

The curve defines the value (Y axis) of the property over time (X axis).

See F-Curves.

Keyframes

Keyframes are user-defined values on certain frames and are represented by little black squares which become orange if selected.

See Keyframes for more information.

Handles

Each keyframe has a handle that helps determine the values of the curve between keyframes. These handles are represented by extruding lines with circular ends and can be selected and modified to change the shape of the curve.

See Handle Types for more information.

A simple curve.¶

See also

See F-Curves for more info.

Navigation¶

As with most editors, you can:

Pan

Pan the view vertically (values) or horizontally (time) with click and drag MMB.

Zoom

Zoom in and out with the mouse wheel Wheel.

Scale View

Scale the view vertically or horizontally Ctrl-MMB.

In addition, you can also use the scrollbars to pan and zoom the view.

Playhead & 2D Cursor¶

Graph Editor 2D Cursor.¶

The current frame is represented by a blue vertical line called the Playhead.

As in the Timeline, you can change the current frame by LMB-dragging in the scrubbing area at the top of the editor.

The blue horizontal line is called the 2D Cursor. This can be enabled or disabled via the View Menu or the View Properties panel.

These two lines can be used as a reference for moving and scaling keyframe handles.

See also

See Graph Editor’s View Tab.

View Axes¶

For Actions the X axis represents time, the Y axis represents the value to set the property.

Depending on the selected curves, the values have different meaning: for example rotation properties are shown in degrees.

Preview Range¶

Set Preview Range P

Interactively define frame range used for playback. Allows you to define a temporary preview range to use for animation playback (this is the same thing as the Playback Range option of the Timeline editor header).

Clear Preview Range Alt-P

Clears the preview range.

Auto-Set Preview Range Ctrl-Alt-P

Automatically sets the preview range to playback the whole action or the selected NLA strips.

Markers Menu¶

Markers are used to denote frames with key points or significant events within an animation. Like with most animation editors, markers are shown at the bottom of the editor.

Markers in animation editor.¶

For descriptions of the different marker tools see Editing Markers.

View Controls¶

View controls.¶

Show Only Selected

Only include curves related to the selected objects and data.

Show Hidden

Include curves from objects/bones that are not visible.

Show Only Errors

Only include curves and drivers that are disabled or have errors. Useful for debugging.

Filter popover (funnel icon)

Only include curves with keywords contained in the search field.

Multi-Word

Fuzzy/Multi-Word name filtering matches word snippets/partial words, instead of having to match everything. It breaks down the search string based on white-space placement. e.g. “lo ro” will filter all location and rotation, while “lc rt” will not work.

Type Filter

Filter curves by property type.

Filtering Collection

Select a collection to only show keyframes from objects contained in that collection.

Sort Data-Blocks

Objects data-blocks appear in alphabetical order, so that it is easier to find where they occur (as well as helping to keep the animation of related objects together in the NLA for instance).

If you find that your playback speed suffers from this being enabled (it should only really be an issue when working with lots of objects in the scene), you can turn this off.

Normalize

Normalize curves so the maximum or minimum point equals 1.0 or -1.0.

Auto

Automatically recalculate curve normalization on every curve edit. This is useful to prevent curves from jumping after tweaking it.

Pendahuluan ¶

Editor Grafik memungkinkan pengguna menyesuaikan kurva animasi dari waktu ke waktu untuk properti yang dapat dianimasikan. F-Curves .

Editor Grafik. ¶

Wilayah Utama ¶

Tampilan kurva memungkinkan Anda untuk melihat dan mengedit kurva F. Kurva F memiliki beberapa bagian utama:

Kurva

Kurva menentukan nilai (sumbu Y) properti dari waktu ke waktu (sumbu X).

Lihat Kurva F .

Bingkai Utama

Bingkai utama adalah nilai buatan pengguna pada bingkai tertentu dan diwakili oleh kotak hitam kecil yang menjadi oranye jika dipilih.

Lihat Keyframes untuk informasi lebih lanjut.

Menangani

Setiap bingkai utama memiliki pegangan yang membantu menentukan nilai kurva di antara bingkai utama. Pegangan ini diwakili oleh garis ekstrusi dengan ujung melingkar dan dapat dipilih dan dimodifikasi untuk mengubah bentuk kurva.

Lihat Jenis Pegangan untuk informasi lebih lanjut.

Rentang Pratinjau ¶

Setel Rentang Pratinjau P

Menentukan rentang bingkai secara interaktif yang digunakan untuk pemutaran. Memungkinkan Anda menentukan rentang pratinjau sementara yang akan digunakan untuk pemutaran animasi (ini sama dengan opsi Rentang Pemutaran ituTajuk editor garis waktu ).

Hapus Rentang Pratinjau Alt-P

Menghapus rentang pratinjau.

Atur Otomatis Rentang Pratinjau Ctrl-Alt-P

Secara otomatis menyetel rentang pratinjau untuk memutar ulang seluruh aksi atau strip NLA yang dipilih.

Menu Penanda ¶

Penanda digunakan untuk menunjukkan bingkai dengan poin-poin utama atau peristiwa penting dalam sebuah animasi. Seperti kebanyakan editor animasi, penanda ditampilkan di bagian bawah editor.

Lihat Kontrol¶

Lihat kontrol. ¶

Hanya Tampilkan yang Dipilih

Hanya menyertakan kurva yang terkait dengan objek dan data yang dipilih.

Tampilkan Tersembunyi

Sertakan kurva dari objek / tulang yang tidak terlihat.

Tampilkan Hanya Kesalahan

Hanya sertakan kurva dan driver yang dinonaktifkan atau memiliki kesalahan. Berguna untuk debugging.

Popover filter (ikon corong)

Hanya menyertakan kurva dengan kata kunci yang terdapat di bidang pencarian.

Multi-Kata

Pemfilteran nama Fuzzy / Multi-Kata cocok dengan cuplikan kata / sebagian kata, alih-alih harus mencocokkan semuanya. Ini memecah string pencarian berdasarkan penempatan ruang putih. misalnya “Lo ro” akan memfilter semua lokasi dan rotasi, sedangkan “lc rt” tidak berfungsi.

Jenis Filter

Memfilter kurva menurut jenis properti.

Memfilter Koleksi

Pilih koleksi untuk hanya menampilkan bingkai utama dari objek yang ada dalam koleksi itu.

Urutkan Blok Data

Blok-data objek muncul dalam urutan abjad, sehingga lebih mudah untuk menemukannya terjadi (serta membantu menjaga animasi objek terkait bersama-sama di NLA untuk misalnya).

Jika Anda menemukan bahwa kecepatan pemutaran Anda berkurang karena pengaktifan ini (ini seharusnya hanya menjadi masalah saat bekerja dengan banyak objek di tempat kejadian), Anda bisa mematikannya.

Normalisasi

Normalisasikan kurva sehingga titik maksimum atau minimum sama dengan 1.0 atau -1.0.

Otomatis

Menghitung ulang normalisasi kurva secara otomatis pada setiap pengeditan kurva. Ini berguna untuk mencegah kurva melompat setelah menyesuaikannya.
