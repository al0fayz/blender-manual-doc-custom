Grease Pencil¶

This Mode allows you adjust the timing of the Grease Pencil object’s animation frames. It is especially useful for animators blocking out shots, where the ability to re-time blocking is one of the main purposes of the whole exercise.

This mode can be accessed by changing the Dope Sheet editor’s Mode selector (found in the header to the far left) to Grease Pencil.

To use this editor mode, make sure you have a Grease Pencil object selected.

Channels Region¶

Grease Pencil (light blue)

The channels’ region shows the Grease Pencil data-blocks containing the layers. Multiple blocks are used for each area (e.g. one for the 3D Viewport and the Image editor).

Layers (gray)

These channels contain the keyframes to which the layers are bound.

Opacity

Controls the layers Opacity.

Mask (mask icon)

Toggle the Masks visibility.

Onion Skinning (onion skin icon)

Toggle the use the layer for Onion Skinning.

Viewport/Render Visibility (eye icon)

Toggle layer visibility in the viewport and in render.

Lock Animation (checkbox icon)

Toggles playback of animation or keep the channel static.

Lock (padlock icon)

Toggle layer from being editable.

Header¶

Layer Move

Moves the selected layer/channel up or down in the evaluation stack.

Layer Add/Remove

Adds/removes Grease Pencil layers/channels.

Insert Keyframe¶

Insert Keyframe I can be used for creating blank Grease Pencil frames at a particular frame. It will create blank frames if Additive Drawing is disabled, otherwise it will make a copy of the active frame on that layer, and use that.

Copying Frames¶

It is possible to copy frames from one layer to another, or from object to object, using the Copy and Paste tools in the Key menu. Note that keyframes will be pasted into selected layers, so make sure you have a destination layer selected.

Main View¶

The keyframes can be manipulated like any other data in the Dope Sheet. Interpolated keyframes (alias breakdowns) are visualized as smaller light blue points.

Sidebar¶

The Sidebar contains a copy of the Grease Pencil Layer Properties.

Pensil Gemuk ¶

Mode ini memungkinkan Anda menyesuaikan pengaturan waktu Objek Pensil Gemuk bingkai animasi. Ini sangat berguna untuk animator yang memblokir tembakan, di mana kemampuan untuk mengatur waktu ulang pemblokiran adalah salah satu tujuan utama dari keseluruhan latihan.

Mode ini dapat diakses dengan mengubah pemilih Mode editor Dope Sheet (ada di header ke paling kiri) ke Grease Pencil.

Untuk menggunakan mode editor ini, pastikan Anda memiliki Objek Pensil Gemuk dipilih.

Wilayah Saluran ¶

Grease Pencil (biru muda)

Wilayah saluran menampilkan blok data Pensil Gemuk yang berisi lapisan. Beberapa blok digunakan untuk setiap area (mis. Satu untuk 3D Viewport dan editor Gambar).

Lapisan (abu-abu)

Kanal ini berisi bingkai utama lapisan terikat.

Opasitas

Mengontrol lapisan Opasitas .

Topeng (ikon topeng)

Alihkan ke Masker visibilitas.

Menguliti Bawang (ikon kulit bawang)

Alihkan penggunaan lapisan untukBawang Menguliti .

Viewport / Render Visibility (ikon mata)

Alihkan visibilitas lapisan di viewport dan di render.

Animasi Kunci (ikon kotak centang)

Beralih pemutaran animasi atau menjaga saluran tetap statis.

Kunci (ikon gembok)

Alihkan lapisan agar tidak dapat diedit.

Tajuk ¶

Pemindahan Lapisan

Memindahkan lapisan / saluran yang dipilih ke atas atau ke bawah dalam tumpukan evaluasi.

Tambah / Hapus Lapisan

Menambahkan / menghapus lapisan / saluran Pensil Gemuk.

Sisipkan Bingkai Utama ¶

Sisipkan Bingkai Utama I dapat digunakan untuk membuat Kosong bingkai Pensil Gemuk pada bingkai tertentu. Ini akan membuat bingkai kosong jika Gambar Aditif dinonaktifkan, jika tidak itu akan membuat salinan bingkai aktif pada lapisan itu, dan menggunakannya.

Menyalin Bingkai ¶

Memungkinkan untuk menyalin bingkai dari satu lapisan ke lapisan lainnya, atau dari satu objek ke objek lain, menggunakan alat Salin dan Tempel di Kunci Tidak bisa. Perhatikan bahwa bingkai utama akan ditempelkan ke lapisan yang dipilih, jadi pastikan Anda memiliki lapisan tujuan dipilih.

Tampilan Utama ¶

Bingkai utama dapat dimanipulasi seperti data lainnya di Dope Sheet . Keyframe yang diinterpolasi (pengelompokan alias) divisualisasikan sebagai titik biru muda yang lebih kecil.

Bilah sisi ¶

Bilah Sisi berisi salinan Grease Pencil Properti Lapisan .
