Input¶

In the Input preferences, you can customize how Blender reacts to the mouse and keyboard as well as define your own keymap.

Keyboard¶

Emulate Numpad

The Numpad keys are used quite often in Blender and are not assigned to the same action as the regular number keys. If you have a keyboard without a Numpad (e.g. on a laptop), you can tell Blender to treat the standard number keys as Numpad keys by checking Emulate Numpad.

Default to Advanced Numeric Input

For transform mode, default to Advanced Mode, otherwise Simple Mode is used.

Mouse¶

Emulate 3 Button Mouse

Blender can be configured to work with pointing devices which do not have a MMB. The functionality of the three mouse buttons by holding Alt-LMB.

Mouse/Keyboard combinations referenced in this manual can be expressed with the combinations shown in the table. For example:

MMB drag becomes Alt-LMB drag for example.

Warning

This option prevents certain features from being accessed, since Alt-LMB is used for some operations.

  • Modifying multiple items values at once (objects, bones… etc).

  • Deselecting edge/face rings in Edit Mode.

  • Detaching node links.

  • Moving the Compositor background image.

Some touchpads support three-finger tap for middle mouse, which may be an alternative to using this option.

Modifier (unsupported on Microsoft Windows)
Alt

Use the Alt key to emulate the middle mouse button.

OSKey

Use the OSKey to emulate the middle mouse button.

This has the advantage that it doesn’t conflict with existing Alt-MMB shortcuts, noted above.

Continuous Grab

This feature is used to prevent the problem where an action such as moving objects or panning a view, is limited by your screen bounds.

This is done by warping the mouse within the view.

Note

Cursor warping is only supported by relative input devices (mouse, trackball, trackpad).

Graphics tablets, however, typically use absolute positioning, this feature is disabled when a tablet is being used.

This is detected for each action, so the presence of a tablet will not disable Continuous Grab for mouse cursor input.

Release Confirms

Dragging LMB on an object will move it. To confirm this (and other) transform, an LMB is necessary by default. When this option is activated, the release of LMB acts as confirmation of the transform.

Mouse Drag Threshold

The number of pixels that a User Interface element has to be moved before it is recognized by Blender, values below this will be detected as click events.

Tablet Drag Threshold

The drag threshold for tablet events.

Drag Threshold

The drag threshold for non mouse/tablet events (keyboard or NDOF for example).

This affects Pie Menu on Drag keymap preference.

Motion Threshold

The number of pixels the cursor must be moved before the movement is registered. This is helpful for tablet pens that are a lot more difficult to keep still, then this could help to reduce stuttering of the cursor position.

Note

Unlike the click/drag distinction, this is used to detect small movements for example, picking selection cycles through elements near the cursor. Once the cursor moves past this threshold, selection stops cycling and picks the closest item.

Tablet¶

Tablet API (Windows only)

Select the native Windows Ink or older Wintab system for pressure sensitivity. Blender automatically selects the API for your operating system and tablet, however in case of problems this can be set manually.

Max Threshold

Amount of pressure required to achieve full intensity.

Softness

Controls how the softness of the low pressure response onset using a gamma curve.

NDOF¶

These preferences control how a NDOF device interacts with the 3D Viewport. These preferences can also be accessed using the NDOFMenu button on the NDOF device to open a pop-up menu to adjust the settings directly from the 3D Viewport.

Pan Sensitivity

The overall sensitivity for panning in the 3D Viewport.

Orbit Sensitivity

The overall sensitivity for orbiting in the 3D Viewport.

Deadzone

The threshold for the amount of movement needed from the device’s rest position for Blender to interrupt that movement.

Navigation

Navigation style for the viewport.

Free

Uses the full 6-degrees of freedom.

Orbit

Orbit about the view center.

Rotation

Rotation style for the viewport.

Turntable

Rotates the view keeping the horizon horizontal.

Trackball

Is less restrictive, allowing any orientation.

Show Navigation Guide

Display the pivot point and axis during rotation.

Invert Zoom

Zoom using opposite direction.

Swap Y and Z Axes

Pan using up/down on the NDOF devices instead of forward/backwards.

Invert Axis Pan

Reverses the panning axis on the selected axes.

Orbit

Reverses the orbit axis on the selected axes.

Fly/Walk

Settings to control how the NDOF device is used while using Walk/Fly Navigation.

Lock Horizon

Keeps the horizontal axis level file flying.

Helicopter Mode

Moves the 3D Viewport up or down when moving the NDOF device up/down.

Masukkan ¶

Di preferensi Input, Anda dapat menyesuaikan bagaimana Blender bereaksi terhadap mouse dan keyboard serta menentukan peta kunci Anda sendiri.

Keyboard ¶

Tiru Numpad

Tombol Numpad cukup sering digunakan di Blender dan tidak ditetapkan untuk tindakan yang sama seperti tombol angka biasa. Jika Anda memiliki keyboard tanpa Numpad (misalnya di laptop), Anda dapat memberi tahu Blender untuk memperlakukan tombol angka standar sebagai tombol Numpad dengan mencentang Emulate Numpad .

Default untuk Input Numerik Lanjutan

Untuk mode transformasi, default ke Mode Lanjutan , jika tidak Mode Sederhana digunakan.

Mouse ¶

Meniru 3 Tombol Mouse

Blender dapat dikonfigurasi untuk bekerja dengan perangkat penunjuk yang tidak memiliki MMB . Fungsionalitas ketiga tombol mouse dengan menahan Alt-LMB .

Kombinasi Mouse / Keyboard yang dirujuk dalam manual ini dapat diekspresikan dengan kombinasi yang ditunjukkan pada tabel. Misalnya:

MMB tarik menjadi Alt-LMB seret misalnya.

Peringatan

Opsi ini mencegah fitur tertentu untuk diakses, karena Alt-LMB digunakan untuk beberapa operasi.

  • Memodifikasi beberapa nilai item sekaligus (objek, tulang… dll).

  • Membatalkan pilihan cincin tepi / wajah dalam Mode Edit.

  • Memisahkan tautan node.

  • Memindahkan gambar latar belakang Compositor.

Beberapa touchpad mendukung ketukan tiga jari untuk mouse tengah, yang mungkin merupakan alternatif untuk menggunakan opsi ini.

Modifier (tidak didukung di Microsoft Windows)
Alt

Gunakan tombol Alt untuk meniru tombol tengah mouse.

OSKey

Gunakan OSKey untuk meniru bagian tengah tombol mouse.

Keunggulannya adalah tidak bertentangan dengan Alt-MMB pintasan, disebutkan di atas.

Pengambilan Berkelanjutan

Fitur ini digunakan untuk mencegah masalah dimana suatu tindakan seperti memindahkan objek atau menggeser a melihat, dibatasi oleh batas layar Anda.

Ini dilakukan dengan membelokkan mouse dalam tampilan.

Catatan

Pembengkokan kursor hanya didukung oleh perangkat masukan relatif (mouse, trackball, trackpad).

Tablet grafis, bagaimanapun, biasanya menggunakan pemosisian mutlak , fitur ini dinonaktifkan saat tablet sedang digunakan.

Ini terdeteksi untuk setiap tindakan, jadi kehadiran tablet tidak akan menonaktifkan Continuous Grab untuk masukan kursor mouse.

Rilis Konfirmasi

Menyeret LMB pada objek akan memindahkannya. Untuk mengonfirmasi transformasi ini (dan lainnya), sebuah LMB diperlukan secara default. Saat opsi ini diaktifkan, rilis LMB bertindak sebagai konfirmasi transformasi.

Ambang Tarik Mouse

Jumlah piksel di mana elemen Antarmuka Pengguna harus dipindahkan sebelum dikenali Blender, nilai di bawah ini akan terdeteksi sebagai peristiwa klik.

Ambang Seret Tablet

Ambang tarik untuk acara tablet.

Tarik Ambang

Ambang tarik untuk acara selain mouse / tablet (keyboard atau NDOF untuk contoh).

Ini memengaruhi Menu Pai pada preferensi peta kunci Seret .

Ambang Gerakan

Jumlah piksel yang harus digerakkan kursor sebelum gerakan terdaftar. Ini berguna untuk pena tablet yang jauh lebih sulit untuk diam, maka ini bisa membantu mengurangi gagapnya posisi kursor.

Catatan

Tidak seperti perbedaan klik / seret, perbedaan ini digunakan untuk mendeteksi gerakan kecil misalnya, pemilihan siklus melalui elemen di dekat kursor. Setelah kursor bergerak melewati ambang ini, pemilihan berhenti berputar dan memilih item terdekat.

Tablet ¶

Tablet API (khusus Windows)

Pilih Tinta Windows asli atau sistem Wintab yang lebih lama untuk sensitivitas tekanan. Blender secara otomatis memilih API untuk sistem operasi dan tablet Anda, namun jika terjadi masalah, ini dapat diatur secara manual.

Ambang Maks.

Jumlah tekanan yang dibutuhkan untuk mencapai intensitas penuh.

Kelembutan

Mengontrol bagaimana kelembutan onset respons tekanan rendah menggunakan kurva gamma.

NDOF ¶

Preferensi ini mengontrol bagaimana perangkat NDOF berinteraksi dengan 3D Viewport. Preferensi ini juga dapat diakses menggunakan NDOFMenu tombol di perangkat NDOF untuk membuka menu pop-up untuk menyesuaikan pengaturan langsung dari 3D Viewport.

Sensitivitas Geser

Keseluruhan sensitivitas untuk menggeser di Area Pandang 3D.

Sensitivitas Orbit

Sensitivitas keseluruhan untuk mengorbit dalam 3D Viewport.

Zona Mati

Ambang batas jumlah pergerakan yang dibutuhkan dari posisi istirahat perangkat agar Blender menghentikan gerakan itu.

Navigasi

Gaya navigasi untuk area pandang.

Gratis

Menggunakan 6 derajat penuh kebebasan.

Orbit

Orbit tentang pusat tampilan.

Rotasi

Gaya rotasi untuk area pandang.

Meja Putar

Memutar tampilan dengan menjaga cakrawala tetap horizontal.

Trackball

Tidak terlalu membatasi, mengizinkan orientasi apa pun.

Tampilkan Panduan Navigasi

Menampilkan titik dan sumbu pivot selama rotasi.

Pembalikan Zoom

Zoom menggunakan arah berlawanan.

Tukar Sumbu Y dan Z

Menggeser menggunakan atas / bawah pada perangkat NDOF, bukan maju / mundur.

Balikkan Axis Pan

Membalik sumbu panning pada sumbu yang dipilih.

Orbit

Membalikkan sumbu orbit pada sumbu yang dipilih.

Terbang / Berjalan

Setelan untuk mengontrol bagaimana perangkat NDOF digunakan saat menggunakan Jalan / Terbang Navigasi .

Kunci Horizon

Membuat file tingkat sumbu horizontal terus terbang.

Mode Helikopter

Menggerakkan 3D Viewport ke atas atau ke bawah saat menggerakkan perangkat NDOF ke atas / bawah.
