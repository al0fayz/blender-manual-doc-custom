Color Ramp Widget¶

Color Ramps enables the user to specify a range of colors based on color stops. Color stops are similar to a mark indicating where exactly the chosen color should be. The interval from each of the stops, added to the ramp, is a result of the color interpolation and chosen interpolation method.

Color ramp.¶

Controls¶

Add +

Clicking on this button will add a stop to your color ramp. The stops are added from the last selected stop to the next one, from left to right and they will be placed in the middle of both stops.

Delete -

Deletes the selected color stop from the list.

Specials v

Contains more operators for the color ramp.

Flip Color Ramp

Flips the gradient, inverting the values of the color ramp.

Distribute Stops from Left

Distribute the stops so that every step has the same space to the right. This is mostly useful when used with Constant interpolation mode.

Distribute Stops Evenly

Space between all neighboring stops becomes equal.

Eyedropper (pipette icon) E

An Eyedropper to sample a color or gradient from the interface to be used in the color ramp.

Reset Color Ramp

Resets the color ramp to its default state.

Color Mode

Selection of the color space used for interpolation.

RGB

Blends color by mixing each color channel and combining.

HSV/HSL

Blends colors by first converting to HSV or HSL, mixing, then combining again. This has the advantage of maintaining saturation between different hues, where RGB would de-saturate, this allows for a richer gradient.

Interpolation Options

Enables the user to choose the types of calculations for the color interpolation for each color stop.

RGB
B-Spline

Uses a B-Spline Interpolation for the color stops.

Cardinal

Uses a Cardinal Interpolation for the color stops.

Linear

Uses a Linear Interpolation for the color stops.

Ease

Uses an Ease Interpolation for the color stops.

Constant

Uses a Constant Interpolation for the color stops.

HSV and HSL
Clockwise

Clockwise interpolation around the HSV/HSL wheel.

Counter-Clockwise

Counterclockwise around the HSV/HSL wheel.

Near

Nearest route around the wheel.

Far

Furthest route around the wheel.

HSV and HSL interpolation options.¶

Active Color Stop

Index of the active color stop (shown as a dashed line). Allows you to change the active color when colors may be too close to easily select with the cursor.

Position

This slider controls the positioning of the selected color stop in the range.

Color

Opens a color picker for the user to specify color and Alpha for the selected color stop. When a color is using Alpha, the color field is then divided in two, with the left side showing the base color and the right side showing the color with the alpha value.

Shortcuts¶

  • LMB (drag) moves colors.

  • Ctrl-LMB (click) adds a new control point.

Widget Color Ramp ¶

Color Ramps memungkinkan pengguna untuk menentukan rentang warna berdasarkan color stop. Penghentian warna mirip dengan tanda yang menunjukkan di mana tepatnya warna yang dipilih seharusnya. Interval dari masing-masing perhentian, ditambahkan ke ramp, merupakan hasil dari interpolasi warna dan memilih metode interpolasi.

Color ramp.¶

Kontrol ¶

Tambahkan +

Mengeklik tombol ini akan menambahkan penghentian pada tanjakan warna Anda. Pemberhentian ditambahkan dari perhentian yang terakhir dipilih ke perhentian berikutnya, dari kiri ke kanan dan mereka akan ditempatkan di tengah-tengah kedua perhentian.

Hapus -

Menghapus penghentian warna yang dipilih dari daftar.

Spesial v

Berisi lebih banyak operator untuk color ramp.

Flip Color Ramp

Membalik gradien, membalik nilai ramp warna.

Distribusikan Berhenti dari Kiri

Distribusikan perhentian sehingga setiap langkah memiliki ruang yang sama di sebelah kanan. Ini sebagian besar berguna saat digunakan dengan mode interpolasi Konstan.

Distribusikan Berhenti Secara Merata

Spasi di antara semua perhentian yang berdekatan menjadi sama.

Pipet (ikon pipet) E

Sebuah Eyedropper untuk mengambil sampel warna atau gradien dari antarmuka untuk digunakan di jalur warna.

Setel Ulang Color Ramp

Menyetel ulang tanjakan warna ke keadaan defaultnya.

Mode Warna

Pemilihan warna spasi digunakan untuk interpolasi.

RGB

Memadukan warna dengan mencampurkan setiap saluran warna dan menggabungkannya.

HSV / HSL

Memadukan warna dengan mengonversinya terlebih dahulu ke HSV atau HSL, mencampur, lalu menggabungkan lagi. Ini memiliki keuntungan dalam menjaga saturasi antara warna yang berbeda, di mana RGB akan mengalami de-saturasi, ini memungkinkan gradien yang lebih kaya.

Opsi Interpolasi

Memungkinkan pengguna untuk memilih jenis kalkulasi untuk interpolasi warna untuk setiap warna berhenti.

RGB
B-Spline

Menggunakan Interpolasi B-Spline untuk penghentian warna.

Kardinal

Menggunakan Interpolasi Kardinal untuk penghentian warna.

Linear

Menggunakan Interpolasi Linear untuk penghentian warna.

Kemudahan

Menggunakan Interpolasi Kemudahan untuk penghentian warna.

Konstan

Menggunakan Interpolasi Konstan untuk penghentian warna.

HSV dan HSL
Searah Jarum Jam

Interpolasi searah jarum jam di sekitar roda HSV / HSL.

Berlawanan Arah Jarum Jam

Berlawanan arah jarum jam di sekitar roda HSV / HSL.

Dekat

Rute terdekat di sekitar kemudi.

Jauh

Rute terjauh di sekitar kemudi.

Opsi interpolasi HSV dan HSL. ¶

Pemberhentian Warna Aktif

Indeks penghentian warna aktif (ditampilkan sebagai garis putus-putus). Memungkinkan Anda untuk mengubah warna aktif ketika warna mungkin terlalu dekat untuk dipilih dengan mudah dengan kursor.

Posisi

Penggeser ini mengontrol posisi penghentian warna yang dipilih dalam rentang.

Warna

Membuka pemilih warna bagi pengguna untuk menentukan warna dan Alfa untuk penghentian warna yang dipilih. Saat warna menggunakan Alpha, kolom warna kemudian dibagi dua, dengan sisi kiri menampilkan warna dasar dan sisi kanan menampilkan warna dengan nilai alpha.

Pintasan ¶

  • LMB (seret) memindahkan warna.

  • Ctrl-LMB (klik) menambahkan kontrol baru poin.
